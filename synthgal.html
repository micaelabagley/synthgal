class synthgal.filters.Filter(filt, wave_array)[source]¶

Bases: object

A class to define a broadband filter curve.

The filter transmission curve is read in, normalized, and interpolated onto a finer-resolution wavelength array. In Filter.calc_synphot(), the spectrum is interpolated to the same wavelength array. The spectrum and the filter used to calcuate the broadband photometry therefore have the same wavelength resolution.

Filter transmission curves are stored in the configs directory and are called [filter].txt. Available HST/WFC3 filters are:

  • F110W
  • F160W
  • F140W
Args:
wave_array (float): Wavelength array for interpolation of
filters and SED templates

class synthgal.template.Template(metallicity, flux, ew, redshift, add_lines, output)[source]¶

Bases: object

A class to read in BC03 templates.

BC03 templates are stored in the synthsrc/highztemplates directory and are called [IMF]_[metal]_[SFH].fits, where

  • IMF is either salp or chab for Salpeter and Chabrier initial mass functions, respectively;
  • metal is the metallicity ranging from m22 to m62 (solar);
  • SFH is one of tau0p01, tau0p50, and tau5p00, for exponentially declining star formation histories with characteristic timescales of 0.01, 0.5, and 5 Gyr, respectively.
